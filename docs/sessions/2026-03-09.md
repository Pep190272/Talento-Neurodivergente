## Sesion PM: Expansion SaaS — Modelo de Negocio + Nuevos Bounded Contexts

### 6. Analisis de modelo de negocio

Investigacion de modelos de referencia (LinkedIn, Mentra, Specialisterne, plataformas de cursos):
- **Modelo aprobado**: SaaS + Marketplace mixto
- **Candidatos**: siempre gratis (son el valor)
- **Empresas**: Free / Starter(49 EUR) / Pro(149 EUR) / Enterprise(399+ EUR)
- **Candidatos B2C**: Free / Pro(9.99 EUR) / Pro+(19.99 EUR)
- **Terapeutas**: Free / Profesional(29 EUR)
- **Pasarela**: Stripe
- **Early adopters**: 6 meses gratis (20 empresas, 50 terapeutas)
- **Breakeven**: Mes 7 (~788 EUR MRR)

### 7. Analisis DDD de servicios existentes

Auditoria completa de los 4 servicios para extraer patrones replicables:
- **Patron de entidad**: Rich Domain Model con factory methods y behavior
- **Patron de repositorio**: Interface (port) en dominio, implementacion en infra
- **Patron de use case**: Orquestacion sin logica de negocio
- **Patron de persistencia**: `to_entity()` / `from_entity()` en cada ORM model
- **Value Objects**: Email, Score, NeuroVector24D — inmutables con validacion
- **Convenciones SQL**: snake_case, VARCHAR(25) IDs, JSONB para nested, Enums con schema
